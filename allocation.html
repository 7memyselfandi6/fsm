Fertilizer Allocation System
1. Total Amount Purchased
Here define the total amount of fertilizer to be allocated. Here the total amount of each fertilizer should be defined for the year selected. And this is the base for further allocation.
2. R-Distn
This button is used to set ratio of each fertilizer type to be allocated to regions. The system helps in calculating the ratio, so that it will be easy to proportionally allocate to each region.
3. Season
Here the fertilizer allocation is made to each region to the different cropping seasons, viz. rainy season, belg season and irrigation season.
4. Calc
This button is used to calculate the regional fertilizer distribution. To know the allocated amount to each region based on the previously fixed ratio, clicking the light green buttons labeled with each cropping seasons is required.
